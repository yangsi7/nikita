## Responsive Breakpoints

**Mobile** (<640px):
- Single column layouts
- Stacked navigation
- Bottom tab bar
- Hamburger menu
- Touch-optimized (44x44px tap targets)

**Tablet** (640px-1024px):
- 2-column grids
- Collapsible sidebar
- Compact navigation
- Card-based layouts

**Desktop** (>1024px):
- Multi-column grids (3-4 columns)
- Persistent sidebar
- Expanded navigation
- Optimized for mouse/keyboard

---

## Notes

- All form inputs should have associated labels
- Error messages appear below form fields
- Loading states should appear during data fetching
- Empty states guide users to take action
- Modals center on screen with backdrop
- Toasts appear top-right, stack vertically
